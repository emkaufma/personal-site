BoardgameDB

BoardgameDB offers a way for users to view games, create groups, events, as well as other activities. The site also has user registration and login functionality. The application was created using the LEMP stack.

The source code for the website can be found on my github.


Linkup

Linkup offers a way for teachers to group up students in a way that remembers previous pairings and history. Future plans are to create a store where teachers can purchase activity cards to facilitate student engagement using the pairing system. This project is front-end AngularJS with an Express back-end and a MySQL database.
